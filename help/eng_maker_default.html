Welcome to Tuitpix's maker tool!

This "maker" is a tool to draw your own pixel art parts by type.

Connect your twitter account with the "t" button to obtain this cool features:
  • Save and load your custom parts.
  • Add them as a new category on the list of parts in the main page so you can use them if you want.
Change the type of part and go back to load the last stored design.

The menu is on the top-left corner.
